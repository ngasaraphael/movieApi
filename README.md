## **movieApi**      
movie api provide details of movies

## Built With.  
MERN Tech stack (Frontend built using React library).   

## **Node dependencies**    
Express.  
morgan for logging.  
bodyParser.  
mongoose.   
cors.   
express-validator.   
passport.   
jsonwebtoken    
bcrypt.
